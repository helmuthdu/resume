# Helmuth's resume

> My resume powered by React and Redux: [https://helmuthdu.github.io/resume/](https://helmuthdu.github.io/resume/)

## Build Setup

``` bash
# install dependencies
yarn

# serve with hot reload at localhost:3000
yarn run start

# build for production with minification
yarn run build

# run tests
yarn test

# deploy
yarn run deploy
```
